Editores de código del 2022

Los editores de código son muy utilizados hoy en día, principalmente en el mundo de la programación, que es donde está enfocado este top. Para empezar, debemos saber que es un editor de código y para qué sirven, un editor de código es un programa que esta destinado a editar archivos de texto fuente de alguna aplicación, sitio, programa, etc. El objetivo de estos editores son el acelerar la escritura de código, teniendo como ayuda como autocompletar, un aspecto visual agradable, además que poder utilizar plugin que dependiendo del mismo agregan características al editor. Los editores que se mencionaran adelante son gratuitos, cada editor tendrá el link para su descarga además de mencionar sus características principales.

Visual studio code

Visual Studio Code es un editor de código fuente creado por Microsoft para Windows, Linux y macOS. Las características incluyen soporte para depuración, resaltado de sintaxis, finalización inteligente de código, fragmentos de código, refactorización de código y Git incrustado.

Sublime

Sublime Text es un editor de código fuente comercial. Es compatible de forma nativa con muchos lenguajes de programación y lenguajes de marcado. Los usuarios pueden ampliar su funcionalidad con complementos, generalmente creados y mantenidos por la comunidad bajo licencias de software libre. Para facilitar los complementos, Sublime Text cuenta con una API de Python.
